# remote_relay
rpi relay remote control


## Setup
  
### Setp1
  setup [google firebase](https://firebase.google.com/docs/web/setup)
### Setp2
  clone repository to your PC and RPi
### Setp3
  set your web dev env
  
  ```
  
  cd <path_to_remote_relay>/web/
  
  ```
### Setp4
  set your pi
  
  ```
  cd <path_to_remote_relay>/client/
  npm install
  node app.js
  ```
  
### Setp5
  wiring
